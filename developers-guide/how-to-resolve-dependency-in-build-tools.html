Resolve wffweb dependency in build tools

Apache Buildr
'com.webfirmframework:wffweb:jar:2.0.1'
Gradle Grails
compile 'com.webfirmframework:wffweb:2.0.1'
Groovy Grape
@Grapes( 
@Grab(group='com.webfirmframework', module='wffweb', version='2.0.1') 
)
ivy
<dependency org="com.webfirmframework" name="wffweb" rev="2.0.1" />
Leiningen
[com.webfirmframework/wffweb "2.0.1"]
maven
<dependencies>
    <dependency>
        <groupId>com.webfirmframework</groupId>
        <artifactId>wffweb</artifactId>
        <version>2.0.1</version>
    </dependency>
</dependencies>
To automatically resolve with the latest version
<dependencies>
    <dependency>
	    <groupId>com.webfirmframework</groupId>
	    <artifactId>wffweb</artifactId>
	    <version>RELEASE</version>
    </dependency>
</dependencies>
